Getting Started

Login to USCIS
100% Privacy-Focused Tracker
Your data never leaves your browser. All case information is stored locally and no data is sent to any server.
  1. First, log in to myaccount.uscis.gov in a separate tab
  2. Add your USCIS receipt numbers below (e.g., IOE1234567890)
  3. Click "Open API tabs" to open official USCIS JSON endpoints
  4. Copy the raw JSON from each tab and paste it into Step 2
  5. Your timeline will update automatically with each snapshot
1

Add Your Case Numbers

Start by adding your receipt number(s)
Enter your USCIS receipt number in the format: 3 letters + 10 digits (e.g., IOE1234567890). You can track multiple cases at once.
Format: 3 letters followed by 10 numbers (e.g., IOE1234567890)
2

Get Case Data from USCIS

How to get your case data
  1. Make sure you're logged into myaccount.uscis.gov
  2. Click "Open API tabs" above to open the official USCIS JSON endpoints
  3. Copy all the JSON text from each tab (it looks like code)
  4. Paste it into the box below and click "Save Snapshot"

Immigration Forms Supported

    

      Our USCIS tracker supports all immigration forms including: I-485 Adjustment of Status (Green Card),
      I-765 Employment Authorization Document (Work Permit/EAD), I-131 Travel Document (Advance Parole/Re-entry Permit),
      I-140 Immigrant Worker Petition, I-130 Family-Based Immigration Petition, N-400 Naturalization Application (Citizenship),
      I-129 Nonimmigrant Worker Petition (H-1B, L-1), I-129F Fiancé Visa, I-90 Green Card Renewal, I-751 Remove Conditions on Residence,
      I-539 Extension/Change of Status, and all other USCIS case types.
